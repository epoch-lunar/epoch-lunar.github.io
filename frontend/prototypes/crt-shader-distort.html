Prototype — CRT tube distortion (WebGL post)

Shader CRT — same trace as scope prototype, distorted in fragment shader

The green trace and graticule are drawn to an offscreen 2D canvas each frame, uploaded as a texture, then a fullscreen quad runs a post shader: inverse-barrel UV (tube curvature), radial chromatic offsets (Shadertoy-style RGB fringe), horizontal slot-mask, scanlines, vignette, and slight scan flicker. Inspired by CRT demos such as 4dlyWX, WsVSzV, and 4sf3Dr. Use http://localhost if WebGL is picky on file://.
