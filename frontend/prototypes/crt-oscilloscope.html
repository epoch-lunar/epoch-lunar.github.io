Prototype — P31-style phosphor CRT (oscilloscope)

Analog display — mock / not calibrated

Green trace and graticule evoke classic analog scopes (P31 phosphor). Persistence is simulated by fading the framebuffer each frame; bloom uses canvas shadow. The curved “tube” is CSS (elliptical face + vignette), not a physically accurate CRT model — good enough for mood and layout reference before a real renderer.
